Claude resuelve bien las tareas donde hay mucho texto de por medio: analizar un informe de cien páginas, redactar con cuidado o programar sobre un proyecto entero. Es para quien trabaja con documentos extensos, escribe contenido que tiene que quedar bien y para programadores que quieren un asistente directo en la terminal con Claude Code.

## Cómo sacarle el jugo, paso a paso

1. Entrá a claude.ai y creá una cuenta gratis. El plan gratis ya sirve para chatear y subir archivos.
2. Subí tus documentos (PDF, texto, imágenes) y pedile análisis sobre ellos. Maneja contexto largo, así que podés darle material extenso de una.
3. Sé explícito con el tono y el formato que querés. Claude responde bien a instrucciones detalladas y a pedidos de estructura.
4. Para escritura, pedile primero un esquema, aprobalo y recién después que desarrolle. Sale mejor que pedir todo de un saque.
5. Si programás, instalá Claude Code en la terminal: lee tus archivos, propone cambios y corre comandos con tu permiso.
6. Cuando una respuesta no te cierra, decile qué falló en concreto en vez de repetir el pedido entero.

## Ejemplos para copiar

```
Te paso un informe en PDF. Quiero un resumen ejecutivo de 200
palabras, los tres riesgos principales en bullets y una tabla
con las cifras clave. No agregues nada que no esté en el documento.
```

```
Revisá este texto y marcá qué frases son ambiguas o redundantes.
Proponé una reescritura, pero explicá el porqué de cada cambio.

[pegás tu texto]
```

## Trucos y errores comunes

- Para análisis de documentos, pedile que cite la parte del texto en la que se basa: baja el riesgo de invención.
- Aprovechá el contexto largo pegando todo el material junto en vez de fragmentarlo en varios mensajes.
- En escritura, el flujo esquema → aprobación → desarrollo da resultados más controlados.
- Como todo modelo, puede equivocarse en datos y fechas; verificá lo que sea factual.
- No le pidas que invente datos que no le diste: si trabaja solo con tu material, marcaselo.

## Cuándo usar esta y cuándo otra

Para textos largos, análisis de documentos y escritura cuidada, Claude es de las opciones más sólidas. Para programar dentro de tu proyecto, Claude Code en la terminal supera a un chat común. Si buscás respuestas con fuentes web actualizadas, usá Perplexity; para tareas rápidas y cotidianas, ChatGPT también cumple.
