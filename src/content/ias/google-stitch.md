Google Stitch resuelve el cuello de botella de validar una idea de producto cuando no sabés diseñar ni programar. Le describís la app que tenés en la cabeza y te genera las pantallas, los componentes y los flujos, devolviendo tanto el diseño visual como el código HTML/CSS. Es compatible con Material Design, así que arranca con una base prolija. Es para emprendedores, gente de producto y desarrolladores que quieren prototipar rápido.

## Cómo sacarle el jugo, paso a paso

1. Entrá a stitch.withgoogle.com con tu cuenta de Google. Se usa desde el navegador.
2. Describí la app: qué hace, qué pantallas necesita y a quién apunta.
3. Generá una primera versión de las pantallas y revisá el flujo entre ellas.
4. Pedí ajustes en lenguaje natural: cambiar colores, reordenar componentes o sumar una pantalla.
5. Copiá el código HTML/CSS que devuelve para empezar a construir o mostrar la maqueta.
6. Iterá variantes de una misma pantalla para comparar y elegir la que mejor funcione.

## Ejemplos / ideas de uso

- Generar el primer prototipo de una app para mostrárselo a un socio o cliente.

```
Diseñá una app de finanzas personales con 3 pantallas: inicio
con saldo y gastos del mes, listado de transacciones y un
formulario para cargar un gasto. Estilo limpio, Material Design.
```

- Explorar variantes de una pantalla de onboarding para decidir cuál convence más.
- Sacar una base de código HTML/CSS para arrancar el desarrollo sin partir de cero.

## Trucos y errores comunes

- Cuanto más concreto el pedido (pantallas, datos, acciones), mejor el resultado; evitá descripciones vagas.
- El código y el diseño son un punto de partida, no una app terminada: revisalos antes de usarlos en serio.
- Aprovechá la base Material Design para mantener consistencia visual.
- Es un experimento de Google Labs: puede cambiar o tener disponibilidad limitada; confirmá en la web oficial.

## Cuándo conviene

Cuando querés validar o mostrar una idea de app rápido y no tenés equipo de diseño, Stitch es lo más directo. Si necesitás generar imágenes o ilustraciones sueltas, mejor un generador de imágenes. Para probar y afinar la lógica de IA detrás de tu producto, pasá por Google AI Studio.
