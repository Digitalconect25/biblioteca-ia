Nano Banana resuelve el problema de generar y editar imágenes sin manejar software de diseño, conversando con el asistente Gemini. Le pedís una imagen o le subís una foto y le decís qué cambiar; entiende instrucciones como "sacale el reflejo" o "ponelo de noche". Es para quien quiere retoques rápidos, piezas con texto en varios idiomas y trabajar dentro del ecosistema de Google.

## Cómo sacarle el jugo, paso a paso

1. Entrá a gemini.google.com o abrí la app de Gemini con tu cuenta de Google. Hay uso gratuito con límites; mirá la web oficial por los cupos.
2. Pedile la imagen describiendo sujeto, contexto y estilo en lenguaje natural, como si le hablaras a una persona.
3. Para editar, subí tu foto y pedí el cambio puntual ("cambiá el cielo a atardecer, dejá todo lo demás igual").
4. Si querés un estilo concreto, sumá una foto de referencia y pedile que aplique ese aspecto a tu imagen.
5. Para texto dentro de la imagen, indicá exactamente qué debe decir y en qué idioma; puede traducir y localizar el texto.
6. Iterá conversando: cada respuesta es una base para el siguiente ajuste, así que pedí un cambio por vez.

## Ejemplos de prompts

```
Generá una ilustración de un puesto de frutas en una feria,
luz de mañana, estilo acuarela, y un cartel que diga
"FRUTAS FRESCAS" en letras grandes.
```

```
Tomá esta foto del producto y cambiá el fondo a un estudio
blanco limpio. Mantené el producto, su color y su posición
exactamente igual.
```

```
Convertí esta foto de día en una escena nocturna con luces
cálidas de ciudad, sin mover a las personas del primer plano.
```

## Trucos y errores comunes

- Para editar, sé explícito con lo que NO debe cambiar; si no, puede rehacer partes que estaban bien.
- Aprovechá el conocimiento de Gemini: pedile contexto real (banderas, mapas, marcas reconocibles) y suele acertar mejor que otros.
- Para texto, mantenelo corto y aclarando el idioma; las frases largas pueden salir con errores.
- Toda imagen lleva marca SynthID invisible y etiqueta de IA; tenelo en cuenta si la vas a publicar.
- Verificá detalles finos antes de usarla en serio: como todo modelo, puede equivocarse en proporciones o texturas.

## Cuándo usar esta y cuándo otra

Nano Banana es muy cómodo para editar fotos conversando y para quien ya vive en el ecosistema de Google. Para calidad artística y control de estilo fino, Midjourney es más fuerte. Si necesitás tipografía y diseño con texto largo, mirá Ideogram. Si querés modelos abiertos para correr en tu propia infraestructura, FLUX es la opción.
