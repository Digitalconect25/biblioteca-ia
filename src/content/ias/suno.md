titulo: Suno
nivel: principiante
resumen: Genera canciones completas (música y voz) a partir de una descripción en texto.
descripcion: >
  Suno es un generador de música por IA. Le describís el estilo, el tema y la letra (o dejás
  que la invente) y crea una canción completa, con instrumentos y voz cantada. Sirve para
  maquetas, jingles, fondos musicales o simplemente para experimentar, sin saber tocar ningún
  instrumento.
comoAcceder: >
  Entrá a suno.com y creá una cuenta. Hay un nivel gratuito con créditos diarios para generar
  canciones y planes de pago para más generaciones y derechos de uso comercial.
gratis: true
planPago: Planes de pago con más créditos y, según el plan, derechos para uso comercial.
funciones:
  - Crear canciones a partir de un texto
  - Generar música con voz cantada
  - Elegir estilo y género
  - Escribir tu propia letra o dejar que la genere
casosDeUso:
  - Crear un jingle o una cortina musical
  - Hacer una canción personalizada para un regalo
  - Generar música de fondo para un video
