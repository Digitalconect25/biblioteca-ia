Perplexity resuelve el problema de no saber si lo que te dice una IA es cierto. En vez de una lista de links o una respuesta sin respaldo, te contesta la pregunta y cita las fuentes con enlaces para que verifiques. Es para quien investiga temas actuales, compara opciones o necesita respuestas que pueda chequear, no solo creer.

## Cómo sacarle el jugo, paso a paso

1. Entrá a perplexity.ai con el navegador o usá la app. El plan gratis ya sirve para investigar.
2. Hacé preguntas completas, no palabras sueltas. "Mejor notebook" rinde menos que "Mejor notebook para edición de video por menos de cierto presupuesto en 2026".
3. Abrí las fuentes citadas y leé las originales. Las citas son el valor de la herramienta: usalas.
4. Encadená preguntas de seguimiento. Mantiene el contexto, así que podés ir afinando sin repetir todo.
5. Cuando quieras acotar, pedile que busque solo en fuentes recientes o en un tipo de sitio concreto.
6. Para comparaciones, pedí una tabla con columnas claras y la fuente de cada dato.

## Ejemplos para copiar

```
¿Qué cambios introdujo la última actualización de [tema]?
Resumí en bullets y citá la fuente oficial de cada punto.
Priorizá información de los últimos meses.
```

```
Compará [opción A] y [opción B] en una tabla: precio, ventajas,
desventajas y para qué tipo de usuario conviene cada una.
Indicá la fuente de cada fila.
```

## Trucos y errores comunes

- Leé siempre las fuentes citadas: la IA puede resumir mal aunque el enlace sea correcto.
- Para temas que cambian rápido, pedile explícitamente información reciente.
- No lo uses para escritura creativa larga o código complejo; está pensado para investigar, no para redactar de cero.
- Una pregunta vaga trae fuentes vagas. Cuanto más específica, mejores los enlaces.
- Si una respuesta no cita nada, desconfiá y reformulá pidiendo fuentes.

## Cuándo usar esta y cuándo otra

Para investigar, comparar y verificar con fuentes, Perplexity le gana a un chatbot común. Para escribir, generar ideas o programar, ChatGPT o Claude son mejores. Si querés saber qué se está comentando ahora mismo en redes, Grok con su acceso en tiempo real es la opción.
