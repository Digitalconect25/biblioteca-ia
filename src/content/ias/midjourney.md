Midjourney resuelve el problema de crear imágenes de alta calidad sin saber dibujar ni manejar software de diseño. Describís en texto lo que querés y genera ilustraciones, conceptos o piezas para redes con un acabado artístico reconocible. Es para diseñadores, creadores de contenido y cualquiera que quiera explorar ideas visuales, con la condición de aprender a describir bien.

## Cómo sacarle el jugo, paso a paso

1. Entrá a midjourney.com, creá la cuenta y trabajá desde el editor web. Requiere suscripción de pago; revisá la web oficial para los planes.
2. Empezá el prompt por el sujeto principal, después el contexto, después el estilo. El orden importa: lo primero pesa más.
3. Sumá detalles de iluminación, encuadre y atmósfera. "Un perro" da poco; "un perro viejo durmiendo junto a una ventana, luz de tarde, fotografía analógica" da mucho.
4. Definí la proporción con el parámetro de aspecto según para qué la necesitás (vertical para historias, horizontal para banners).
5. Generá varias opciones, elegí la que más se acerca y pedí variaciones sobre esa antes de reescalar.
6. Iterá el prompt en lugar de reescribirlo entero: cambiá un detalle por vez para entender qué afecta cada palabra.

## Ejemplos para copiar

```
retrato de una mujer mayor sonriendo, piel con textura realista,
luz suave de ventana, fondo desenfocado, fotografía 85mm,
tonos cálidos --ar 3:4
```

```
ilustración plana de una ciudad futurista al atardecer,
estilo vectorial, paleta púrpura y naranja, formas geométricas
simples, sin texto --ar 16:9
```

## Trucos y errores comunes

- Usá referencias de estilo ("estilo acuarela", "fotografía analógica", "render 3D") en vez de adjetivos vagos como "lindo".
- Indicá lo que NO querés con la opción de negativos si un elemento aparece de más.
- Evitá prompts demasiado largos y contradictorios: muchas instrucciones que se pelean entre sí confunden el resultado.
- Para coherencia entre imágenes, reutilizá los mismos términos de estilo y la misma proporción.
- No esperes texto legible dentro de las imágenes: los generadores suelen fallar ahí.

## Cuándo usar esta y cuándo otra

Para imágenes con calidad artística y control de estilo, Midjourney es de las más fuertes. Si necesitás generar imágenes gratis o dentro del ecosistema de Google, mirá Gemini. Si lo tuyo es texto, investigación o código, esto no aplica: usá ChatGPT, Claude o Perplexity según la tarea.
