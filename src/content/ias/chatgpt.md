ChatGPT resuelve el problema de "no sé cómo empezar". Sirve para escribir, resumir, traducir, programar o entender un tema, y le hablás en lenguaje natural sin aprender comandos. Es la puerta de entrada más usada a la IA, pensada tanto para alguien que nunca tocó una herramienta así como para quien ya la usa todos los días.

## Cómo sacarle el jugo, paso a paso

1. Creá una cuenta gratis en chat.openai.com o bajá la app. El plan gratis ya alcanza para la mayoría de las tareas.
2. Dale contexto: quién sos, para qué necesitás la respuesta y a quién va dirigida. "Escribí un correo" rinde mucho menos que "Escribí un correo formal a un cliente que no pagó la factura, tono firme pero amable".
3. Pedí un formato concreto: lista, tabla, pasos numerados o cantidad de palabras.
4. Iterá. La primera respuesta es un borrador. Decile "más corto", "más técnico" o "dame tres versiones" y se ajusta.
5. Para tareas largas, subí archivos (PDF, imágenes, planillas) y pedile que trabaje sobre ellos.
6. Si una conversación se desvía, abrí un chat nuevo: el historial viejo lo confunde.

## Ejemplos para copiar

```
Actuá como editor. Te paso un párrafo y quiero tres versiones:
una formal, una neutra y una para redes. Corregí errores y
explicá en una línea qué cambiaste en cada una.

[pegás tu párrafo]
```

```
Explicame qué es una API como si tuviera 15 años, con una
analogía de la vida real y un ejemplo concreto. Máximo 150 palabras.
```

## Trucos y errores comunes

- Asignale un rol ("actuá como abogado/profesor/programador") para subir la precisión del tono.
- Pedile que muestre su razonamiento paso a paso cuando es un problema de lógica o cálculo.
- No confíes en datos, fechas o citas sin verificar: puede inventar con seguridad. Pedí fuentes y contrastalas.
- Evitá prompts de una sola palabra. El detalle que ponés vos es el que define la calidad.
- No pegues datos sensibles o privados que no quieras que queden en el historial.

## Cuándo usar esta y cuándo otra

Para escritura general, ideas y tareas cotidianas, ChatGPT es la opción más versátil. Si necesitás respuestas con fuentes citadas y verificables, Perplexity te sirve mejor. Para textos muy largos, análisis de documentos extensos o programar en tu terminal, mirá Claude.
