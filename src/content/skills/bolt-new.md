Bolt.new, de StackBlitz, construye aplicaciones web full-stack a partir de texto y las corre en un entorno de desarrollo completo dentro del navegador. La IA escribe y ejecuta el código en vivo, instala dependencias y muestra la app funcionando, y vos podés meter mano en el código cuando haga falta. Sirve para quien quiere la velocidad de generar con IA pero también el control de un editor de verdad, sin instalar nada en su máquina.

## Cómo empezar

1. Entrá a [bolt.new](https://bolt.new) y, si querés guardar tu trabajo, creá una cuenta.
2. Escribí en el chat la app que querés. Bolt arma el proyecto completo en el entorno del navegador.
3. Mirá la vista previa en vivo mientras la IA escribe y ejecuta el código.
4. Iterá conversando o editá el código directamente en el editor integrado.
5. Conectá un backend (base de datos, auth, hosting) si lo necesitás, e importá desde GitHub o Figma cuando te sirva. Publicá cuando esté listo.

## Ejemplo real

Un prompt para arrancar un proyecto concreto:

```
Creá una app de lista de tareas (to-do) con React y TypeScript.
- Agregar, marcar como completada y borrar tareas.
- Que las tareas se guarden para que no se pierdan al recargar.
- Filtro: todas / pendientes / completadas.
- Diseño minimalista con modo claro y oscuro.
```

Después iterás: "agregá una barra de progreso con el porcentaje de tareas completadas".

## Trucos y errores comunes

- Aprovechá que es un entorno real: si la IA se traba, abrí el código y corregilo a mano.
- Importar desde GitHub o Figma te ahorra arrancar de cero cuando ya tenés algo.
- Pedí el proyecto por partes; los pedidos enormes generan más errores que iterar de a poco.
- El uso gratuito tiene límites; las iteraciones largas los consumen rápido.

## Cuándo conviene

Conviene cuando querés generar con IA pero sin perder acceso al código y al entorno de desarrollo, todo en el navegador. Es ideal para quien ya programa y busca acelerar, o para quien quiere aprender viendo el código real que se genera. Si preferís quedarte en lo conversacional sin tocar código, Lovable es más cómodo; si tu foco es UI en React dentro del ecosistema Vercel, mirá v0.
