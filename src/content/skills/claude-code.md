  Claude Code es una herramienta de línea de comandos donde Claude trabaja como un
  programador: lee tu proyecto, edita archivos, corre comandos y te ayuda a construir
  software conversando en lenguaje natural. Es la base sobre la que corren skills y MCPs.

  Le hablás en lenguaje natural ("agregá una página de contacto", "arreglá este error")
  y Claude propone y aplica los cambios, pidiéndote permiso para acciones sensibles.
comoAcceder: >
  Se instala como CLI y también está en la app de escritorio, la web (claude.ai/code) y
  extensiones de editores. Requiere una cuenta de Anthropic.

  # Iniciar Claude Code en tu proyecto

construir o arreglar software sin escribir cada línea a mano.
