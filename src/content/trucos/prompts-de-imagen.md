Pedir "un gato" deja todas las decisiones en manos del generador y te devuelve algo genérico. Cada palabra que agregás es una decisión que tomás vos en lugar de la IA: el sujeto, el estilo, desde dónde se mira la escena y cómo está iluminada. Pensar el prompt como una ficha técnica, y no como una frase suelta, es lo que separa una imagen al azar de una imagen con intención.

## Cómo aplicarlo

1. **Definí el sujeto con detalle.** No "un gato" sino "un gato naranja durmiendo enroscado en el alféizar de una ventana".
2. **Elegí el estilo.** Acuarela, foto realista, ilustración plana, 3D, óleo. El estilo cambia todo el resultado.
3. **Indicá el encuadre.** Plano cercano, plano general, vista cenital, primer plano del rostro. Le decís a la cámara dónde pararse.
4. **Describí la luz.** "Luz cálida de atardecer", "luz dura de mediodía", "neón nocturno". La iluminación define el clima.
5. **Fijá la proporción si importa.** 16:9 para fondos, 1:1 para redes, 9:16 para historias.

## Antes y después

Genérico:

```
un gato en una ventana
```

Con ficha técnica:

```
gato naranja durmiendo enroscado en el alféizar de una ventana,
estilo acuarela suave, plano cercano,
luz cálida de atardecer entrando de costado, proporción 16:9
```

## Llevalo más lejos

- **Negativos.** En generadores que lo permiten, indicá qué NO querés ("sin texto", "sin manos deformes") para limpiar el resultado.
- **Referencias de cámara y lente.** Términos como "lente 50mm", "gran angular" o "poca profundidad de campo" dan control fotográfico fino.
- **Iterá sobre una imagen.** Cuando una se acerca, pedí variaciones cambiando una sola variable por vez (solo la luz, solo el estilo) para entender qué aporta cada palabra.
