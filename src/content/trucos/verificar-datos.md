Los modelos a veces "alucinan": inventan datos con un tono tan convencido que parecen ciertos. No mienten a propósito; es cómo funcionan, predicen texto que suena correcto aunque el dato no exista. El problema no es que se equivoquen, sino que no avisan. Por eso, para cualquier cosa que tenga consecuencias (un precio, una fecha, una ley, una cita, una dosis), la IA te sirve para el borrador, pero la confirmación la hacés vos.

## Cómo aplicarlo

1. **Pedí la fuente de cada dato.** "Al lado de cada afirmación, poné de dónde la sacaste con enlace." Si no puede citar nada, ya es una señal.
2. **Dale permiso para dudar.** "Si no estás seguro de algo, decímelo en vez de inventar." Reduce mucho las invenciones.
3. **Usá una IA con buscador para datos actuales.** Herramientas como Perplexity buscan en la web y muestran las fuentes; sirven más que un modelo que responde de memoria.
4. **Verificá vos lo crítico.** Abrí la fuente y confirmá. La IA acerca el dato; el chequeo final es tuyo.

## Antes y después

Pedido que invita a inventar:

```
¿Cuánto cuesta tramitar un pasaporte en Argentina?
```

Pedido que obliga a respaldar:

```
¿Cuánto cuesta hoy tramitar un pasaporte en Argentina?
Dame el dato con la fuente oficial y el enlace.
Si no tenés información actual y verificable, decímelo
en vez de estimar.
```

## Llevalo más lejos

- **Contrastá dos modelos.** Pasá el mismo dato por dos IAs distintas; donde difieren, ahí mirás con lupa.
- **Pedile que se critique.** "Revisá tu respuesta y marcá qué partes podrían estar mal o desactualizadas."
- **Subí la fuente vos.** Si tenés el documento o la página, pegásela y pedí que responda solo con base en ese material. Así reducís el margen de invención casi a cero.
